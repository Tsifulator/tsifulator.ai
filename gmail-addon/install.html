tsifl

AI Assistant for Financial Analysts — works on any webpage

Quick Install (Developer Mode)

1

Download the extension

Click the button below to download the extension package.

2

Open Chrome Extensions

Navigate to chrome://extensions in Chrome, or go to Menu → Extensions → Manage Extensions.

3

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner.

4

Load the extension

Click Load unpacked and select the unzipped tsifl-extension folder.

5

Pin it

Click the puzzle icon in Chrome's toolbar and pin tsifl for quick access.

Usage

Click the tsifl icon in your toolbar, or use the keyboard shortcut:

Cmd+Shift+T (Mac)   Ctrl+Shift+T (Windows/Linux)

The AI sidebar opens alongside any webpage. It automatically detects Gmail, Google Sheets, Docs, and Slides for context-aware assistance.
